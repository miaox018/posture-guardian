┌─────────────────────────────────────────┐

├─────────────────────────────────────────┤
│ 检测到不良姿势                          │

```

### 4.2 优化后的渐进式提醒层级
```
智能场景识别：
├── 🖥️ 电脑工作模式
│   ├── 第一级：图标变色 (10秒观察)
│   ├── 第二级：轻柔弹窗 (1分钟观察)  
│   └── 第三级：声音提醒 + 5分钟冷却
└── 📚 学习阅读模式
    ├── 第一级：图标变色 (10秒观察)
    └── 第二级：直接声音提醒 + 5分钟冷却 (跳过弹窗)
```

┌─────────────────────────────┐

```

## 5. 响应式设计考虑

### 5.1 弹窗尺寸适配
- 最小宽度: 280px
- 最大宽度: 400px  
- 高度: 自适应内容，最大600px
- 支持滚动（内容过多时）

### 5.2 多语言支持
- 中文（简体）- 默认
- English
- 日本語
- 한국어

## 6. 交互动画设计

### 6.1 加载动画
- 环境光检测：波纹扫描动画
- 姿势分析：骨骼点连接动画
- 数据同步：进度条脉冲

### 6.2 状态转换
- 图标状态切换：0.3s缓动过渡
- 弹窗内容切换：淡入淡出效果
- 提醒出现：从右上角滑入

## 7. 无障碍设计

### 7.1 可访问性支持
- 支持键盘导航
- 高对比度模式
- 声音提醒替代视觉提醒
- 文字大小可调节

### 7.2 色盲友好
- 状态除颜色外增加图标区分
- 重要信息不仅依赖颜色传达

这个原型图设计涵盖了从首次使用到日常监控的完整用户流程，界面简洁易用，符合Chrome扩展的设计规范。每个界面都考虑了用户体验和功能完整性，可以直接用于开发实现。
